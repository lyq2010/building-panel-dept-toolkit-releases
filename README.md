# 建筑板部工具箱

建筑板 / 冷库板项目概览表桌面工具，面向内部项目资料整理使用。它把装箱明细、成品物流、切锯表、下单详情、转账款明细、开票明细等表格自动整理成项目概览表，并刷新「汇总」「每日发货数据」等统计 sheet。

所有业务文件都在本机处理，不上传服务器。本仓库只承接公开下载和发布流程；源码在私有仓维护。

## 下载

到 [Releases 最新版](https://github.com/lyq2010/building-panel-dept-toolkit-releases/releases/latest) 下载：

| 系统 | 文件 | 说明 |
|---|---|---|
| Windows x64 | `packing-list-toolkit_*_x64-setup.exe` | 安装版，支持自动更新 |
| macOS Apple Silicon | `packing-list-toolkit_*_arm64.dmg` | Apple 芯片 Mac 使用 |

Windows 安装版启动后会自动检查新版本；也可以在「关于」里手动检查。macOS 如遇系统安全提示，按公司内部使用规范手动允许打开。

## 典型流程

1. 选择左侧业务入口：「建筑板 · 概览表」或「冷库板 · 概览表」。
2. 把表格拖进对应文件槽，或点击文件槽选择文件。
3. 不上传概览表时，填写项目名和保存目录生成新概览表；上传概览表时，直接更新原概览表。
4. 可按需开启「汇总」「每日发货」「先备份」等选项。
5. 点击「执行」或按 `Ctrl/⌘ + Enter`。
6. 完成后使用「打开文件 / 打开目录」查看结果。

底部回执区会直接显示成功结果或失败原因；右侧事件区保留更细的执行过程，便于排查。

## 业务入口

| 入口 | 生成模式必填 | 更新模式可刷新 | 主要输出 |
|---|---|---|---|
| 建筑板 · 概览表 | 装箱明细、项目名、保存目录 | 概览表、装箱明细、成品物流、切锯表、下单详情、转账款明细、开票明细 | 板材明细、生产/物流状态、合同收款、汇总、每日发货 |
| 冷库板 · 概览表 | 下单详情、项目名、保存目录 | 概览表、下单详情、成品物流、切锯表 | 冷库板明细、状态、汇总、每日发货 |

传入哪类表，就刷新哪一块；未传的可选数据不会被强制要求。更新已有概览表前建议开启「先备份」。

## 从系统拉数

「从系统拉数」可以按下单详情或概览表中的库号、项目名，从生产管理系统拉取：

- 建筑板装箱明细；
- 切锯表；
- 成品物流。

使用限制：

- 需要能联网访问公司生产管理系统；
- 一次只处理一个项目，下单详情或概览表混入多个项目会停止并提示；
- 车间、切锯范围、物流时间窗可在高级设置里调整；
- 物流时间跨度较长时会自动分段查询并合并去重。

## 表格与 Excel

支持 `.xlsx` / `.xlsm`；下单详情额外支持 `.csv`，可识别 UTF-8 或 GBK 编码。

检测到本机 Microsoft Excel 时，工具会优先调用 Excel 做原生重算，确保公式缓存和用户打开文件时看到的结果一致。没有 Excel 时也能生成文件，但公式缓存会等下次用 Excel 打开时重算。

如果输出文件正被 Excel 打开并锁定，工具会在底部回执区提示关闭文件后重试。

## 自动更新

安装版使用 COS 和 R2 双更新通道。GitHub Releases 是人工下载页；自动更新会优先读取发布通道的 `latest.json`。

发布产物包括：

- Windows 安装包；
- macOS DMG；
- macOS 自动更新包；
- `latest.json` 更新清单。
